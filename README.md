A modern, interactive cybersecurity portfolio showcasing my experience, projects, technical skills, certifications, and achievements as a Security Engineer.

## 🚀 Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Lucide React

## ✨ Features

- Responsive modern UI
- Interactive animations
- Cybersecurity project showcase
- Skills & certifications section
- Contact & social media integration
- Optimized performance and clean design
